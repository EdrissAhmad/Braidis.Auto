# Braidis Auto — Website

## Files
- `index.html` — the complete website (single file, no dependencies to install)

## How to Deploy

### Option A: Simple Hosting (Netlify / Vercel / GitHub Pages)
1. Drag and drop `index.html` into netlify.com/drop
2. Your site is live instantly — free hosting, custom domain available

### Option B: cPanel / Shared Hosting
1. Log into your hosting control panel
2. Open File Manager → navigate to `public_html`
3. Upload `index.html`
4. Done — visit your domain

### Option C: Open Locally
Just double-click `index.html` in any browser — it works offline too.

## Features
- Full responsive layout (desktop + mobile)
- Services section (6 services)
- Buy & Sell inventory with:
  - Filter tabs (All / For Sale / Wanted)
  - Book Appointment modal per listing
  - Add new listing form with photo upload
- About section
- Google Reviews section
- Contact form
- Phone call links throughout
